病人哭，不是抗憂鬱劑就能解決——精神科醫師潘建志的「三顆藥」能讓台灣幸福？

沃草／何宇軒（特約記者）、薛翰駿
2015-10-13

「柯P」柯文哲的「白色力量」旋風，在去年1129地方大選橫掃全台，甚至影響了台北市以外的選情；同樣是醫師的柯P競選團隊發言人潘建志，如今乘著這股白色力量旋風，也在明年的立委大選參戰。

[ 採訪側記 ／何宇軒 ]

原本打著柯P「白色力量」旗幟的潘建志，做出加入民進黨的決定，的確產生不小反差。雖然潘自己認為是一種「整合」，但也不免引來「投機」的質疑。

面對藍營佔優勢的選區，到底是要整合在野勢力、接收民進黨的人脈與資源，「先把國民黨拉下來再說」；還是要堅持走自己的路，「監督未來的執政黨」？相信不只是潘建志，也是許多第三勢力候選人面臨的抉擇。

在拜票時，一位民眾說：「我支持『民進黨提名的』潘建志！」這句話具體而微地表現出這樣的難題：面對地方許多重視黨籍更甚於個人的「支持者」，到底要如何爭取選票？

在拜票的過程中，一位活動主持人介紹潘建志的方式更耐人尋味；他說：「他（指潘建志）不是來監督政府，而是來幫小英完－全－執－政－的！」雖然無從得知他當下聽到這句話的心情，但在台下聽眾隨之而來的熱烈掌聲中，或許多少可以感受到，潘建志最後決定棄無黨籍加入民進黨的現實考量吧！

【請簡單介紹自己】

「我是潘建志醫師，柯文哲市長競選團隊的發言人，這次受簡余晏局長的推薦下，參選台北市中山、北松山的立法委員。我從醫23年的期間，發現病患許多小問題都跟政府的立法施政息息相關，如果有機會進入立法院，老百姓的每一分納稅錢我都會好好監督，並盡心盡力來守護人民的健康。」

【為何會想參與選舉，走進體制內？】

「我每天都遇到這些問題，例如有低收入戶來跟我講，『潘醫師，你為什麼要把我的低收入戶資格取消，我要去跳樓』，我跟他講不是我取消的，因為當初他能拿到低收入戶，是我有開精神疾病的證明，所以政府會算分數，分數夠了有一點補助，讓他勉強過日子。結果哪天他的甚麼堂兄又賣了一間老房子，銀行戶頭有錢，他就被取消資格了，他很直接的反應就是跑來罵我，而我心裡很清楚：因為政府沒錢啊，所以會去砍一些人的低收入戶資格。」

「如果不去改變體制，這表示我只是一個醫匠、只會開藥，但是沒辦法解決病人問題；我們精神科對於每個病人的家族史、他的社會功能，經濟狀況都必須了解，我這幾年來感到台灣經濟走下坡，人民非常苦，很多病人坐下來就哭，這不是我開個抗憂鬱劑就能解決的，我看得出來是體制出了問題。」

【你的主要政見是⋯？】

「因為我是醫生，我的文宣就設計成膠囊的形狀，主打『幸福台灣三顆藥』，也就是長照保險、社會住宅、公辦幼托。」

「台灣現在房價所得比世界第一，尤其對台北市的影響是年輕人不敢結婚，因為所得低、不敢成家；不敢結婚之外又不敢生小孩，之後造成台灣人口老化速度世界第一，以後將沒有年輕人來照顧老人。」

/ 長照保險、社會住宅、公辦幼托，救台灣的「三顆藥」/

「所以我提出三顆藥就是針對這些症狀，第一個就是長照保險；長照法在立法院躺了四年才過，那現在長照保險又要另外四年，因為意見非常分歧；國民黨跟小英都有各自的長照保險版本，但內容都太保守，如果照健保的前例來看，一定又會變成血汗長照。」

/ 沒地蓋社會住宅⋯⋯修住宅法解套 /

「住宅法也很重要，柯P說要蓋五萬戶社會住宅、小英說二十萬戶，國民黨出來笑說你沒有地啊！柯P真的沒有地，但是住宅法可以解套，讓地方首長可以徵收中央的用地，來興建只租不售的社會住宅，用這個特別法來提高地方的權限。」

/ 公辦幼托，讓年輕人敢生小孩 /

「那公辦幼托也是跟這個相關，我覺得讓年輕人租房子還是不夠，要讓他們敢生小孩。每年公立幼稚園要開學的時候，每個人都去抽籤，因為學費便宜嘛，沒抽到只好去念私立學費多花好幾萬，所以我們就補助，讓私立的價格跟公立的一樣，就不用去搶公立的，那這個就是花錢啦，或是政府廣蓋公立托兒所也可以，因為很多小學教室都空出來了。」

【認為你的選區最重要的議題是⋯？】

「最主要的就是松山機場要遷建，因為很危險，飛機起降附近全都是高樓大廈。但我著重的是，搬了之後怎麼辦呢？松山機場其實影響到這整塊地的發展非常大，原本全都限高、限建，這些房子都三、四十年，沒人要都更，因為都更起來還是只能蓋到三、四樓，誰要都更啊？松機搬走以後，所有限高、限建、噪音、危險全部解除，這邊馬上就會急速發展，可是到時候誰會先進來？會不會又變成財團的遊樂場？」

「我提出的政見是，以後這塊地全部蓋公園，頂多讓社會住宅或是醫院等公益事業在旁邊。如果全部蓋住宅或商場，台北市又會移入更多人口，但台北市已經不堪負荷了。」

/ Ｑ：不過，松山機場遷建是立委職權嗎？ /

「我會推動松機遷移後的『中央公園特別條例』，規定這邊就是不能亂來；這整塊地幾千億的產值，誘惑太大了，唯一能做的就是訂特別條例把它擋下來，不然甚麼法規都沒辦法擋土地炒作，不過中央要過很難，因為大家都在覬覦這塊地。」

/ Ｑ：要如何跟當地選民說明這項政見？ /

「因為噪音、限建，這些區域每年都拿機場回饋金，但是一個里的回饋金也不過十來萬；如果變中央公園，當限建、限高解除，地價就會漲好幾倍，他們自然就得到土地原本該有的利益，居民如果了解這一塊，自然會去支持。」

/ Ｑ：你如何看待「選民服務」這件事？ /

「我會跟里長、議員、市府密切配合，如果里民來找我們，我們就可以轉介，如果有一些事情是涉及立委職權的，當然會扛下來做。但是我會希望給選民的印象是，我不光是解決當地里民的問題，而是解決全國的問題。所以我們的標語是"Smart Choice"（聰明的選擇），每個人的選擇都會影響到國家，所以選民要聰明一點，不要看到一個年輕人帥帥的就投給他——我就是在說蔣萬安啦。」

【對同選區其他擬參選人的想法】

/ Ｑ：為什麼對蔣萬安是「年輕帥帥」這樣的印象呢？ /

「基本上他跟這塊土地沒有甚麼關聯，也沒辦法有甚麼關聯，因為他是蔣家第四代，就是王孫貴族；他形象好，是因為過去一片空白，甚麼事都沒做過，大家對他了解很少，只要媒體包裝一下就會覺得他不錯。」

「他聲勢會起來，是因為跟羅淑蕾比民調（編按：國民黨的黨內初選），那大家都討厭羅淑蕾，可是對蔣萬安，他進去立法院是好是壞，大家是沒有辦法判斷的，他進去以後投票表決，我想一定是偏黨意、偏財團、偏老國民黨啦！」

/ Ｑ：那你對同選區社民黨李晏榕又有什麼看法？/

「她其實跟蔣萬安滿像的，也是年輕的律師。我很清楚社民黨就是理念型的政黨，所以他們只跟綠黨整合，不跟其他第三勢力整合。」

【說到第三勢力整合，你目前跟民進黨怎麼整合？（*）】

「本來一開始要走柯P模式，就是民進黨先內部整合出一個人，再跟無黨的參選人整合。結果後來梁文傑（原本民進黨在該區提名的參選人，現任台北市議員）退選，民進黨跟我一時都不知道怎麼辦，後來是民進黨主動說，乾脆由他們提名我，作為整合的契機。其實很多人反對這樣，因為這選區之前民進黨都輸，掛民進黨一定不好選，可是不整合一樣選不下去，所以我就接受了。」

「可是我接受之後，民進黨黨內派系又有其他想法，所以現在又要讓我跟阿娟（三立電視台主持人陳斐娟）做民調。細節我也不清楚，但我跟民進黨之間就是用比較低調、配合、謙卑的態度，希望可以好好整合，不然本來就很難打了，如果民進黨沒有支持那根本不用打了啊。」

*採訪當時，陳斐娟尚未宣布退選。至本文完成（10/13），民進黨尚未確定提名潘建志。

/ Ｑ：所以你已經確定加入民進黨了？ /

「我現在已經加入了，這是民進黨的要求，要被徵召就要先入黨，但我大概心裡也有數，雖然要徵召就得入黨，但最後不一定要給我選。」

/ Ｑ：如果在民進黨的民調勝出，你就掛民進黨參選？ /

「我會說是民進黨『徵召』我，因為我們的主文宣早就出來了，所以還是會照原來的方式宣傳，但不會欺騙選民。我會讓選民知道這件事，但也不會一直標榜，只會讓大家知道這是個整合的方式，而我不是被派系或黨意所掌控的。雖然程序上是民進黨提名，但我還是會去爭取中間泛藍選民的支持。」

「如果民調連民進黨內其他人都贏不了，那我再選下去也沒甚麼太大意思，但我相信是不會輸啦！」

/ Ｑ：以民進黨身分參選，是否跟「白色力量」的理念不太相符？ /

「一定會有選民這樣想，但這就要說服選民，這是白色與綠色的結合；我是穿白袍的，我講白色力量沒有錯，我也覺得選民不會因為黨籍，就否定一個人的本質，那我也會去爭取更多死忠民進黨選民支持。我覺得至少過程公開透明，也不會欺騙選民，畢竟我們整合過程就是這麼坎坷嘛，我當然也想要像柯P一樣以無黨籍選啊！」

/ Ｑ：這是否意味著你會受民進黨黨意約束？ /

「就算是小黨，進到立法院一定要跟大黨合作，不然主推的法案沒有一個會過。但進去之後走的方向，不會完全按照黨的意思啦，其實民進黨很多立委也是常常違背黨意的決定，那是國民黨才會完全照黨意。太離譜的政策，就算要把我開除我也抵死不從。」

/ Ｑ：所以雖然你可能代表民進黨參選，但仍會覺得算是一種合作關係？/

「本來就是合作、整合，因為我本來就不是民進黨的啊，是合作的關係啦，我要勝選，就要爭取最多人的支持，理念要感動最多人才能勝選，如果選民只看藍綠就投票，那也沒辦法。」

【你經常提到柯P、強調是他的發言人，為何採取這樣的宣傳方向？】

「因為台北市支持柯P的人有57%，如果那些人願意支持我，那（勝選）就很簡單了，另外我也希望變成柯P在中央的後盾，現在加入民進黨，也希望成為柯P跟民進黨溝通的橋樑。」

/ Ｑ：有網友覺得你是在消費柯P，你怎麼看待？ /

「那種就是酸民的用語，只要曝光就說你愛現、找到任何支持你的人就說你消費，這是國民黨的網軍慣用的模式，可以套到任何人身上，我覺得這沒甚麼太大意義啦！」

「因為評論的程度太低，這什麼人都可以用，像李遠哲如果去幫小英站台，那是不是說小英在消費李遠哲？這樣誰要站台啊？如果網友會這樣認為，就是被網軍耍著玩啦，只要新聞出來就說我在做秀，可是我新聞不出來你哪看得到？」

/ Ｑ：所以你會維持現有的宣傳方式？ /

「當然了，柯P很重要，他也教了我很多事情，他的理念像是公開透明也很重要，我會把立法院所有資料都掏出來，放在網路上，那網友就可以來看為什麼600多個民生法案到現在還沒通過。」

【你最欣賞的台灣政治人物？】

「鄭南榕，我學生時代就認識他了。當時陳永興醫師（現任羅東聖母醫院院長，長期投入黨外運動）是我們精神科的主任，他跟鄭南榕是非常要好的朋友；我自己會當精神科醫師、搞野百合學運，都是受到陳永興醫師的影響。陳永興帶我們到鄭南榕那邊，借用設備來編輯校園地下刊物。」

「當時還在戒嚴，沒人敢在家裡編，也不敢在學校編，最安全的地方就是鄭南榕那邊，就是寫好稿子、編排，最後拿到印刷廠去印刷，再拿到學校來發，他也給我們很多指導。」

「他的理念非常強，為了理想可以獻身；除了敵人攻擊，他連同志背叛都不怕，就是為了理念，一直纏鬥下去。他有時會來開會、跟我們講話，他很愛抽菸，整個牙是黃的（笑），他很熱情，不會覺得我們只是小屁孩就擺架子；他也很有行動力，很多事情我覺得他其實已經想得很清楚了，所以他有那個勇氣，其實主要是對理想堅持的態度吧。」

【選贏，是為了鄭南榕】

「鄭南榕出殯的時候，一萬多人送他到凱道，我們幾個同學一邊哭一邊送他走，那時我都哭死了；走到終點以後，又一道火光冒起來，就詹益樺自焚啊，我就在現場，都快暈倒了，那對我來講是PTSD（創傷後症候群），對我們那代的人而言，真的是刻苦銘心。」

「很重要的是，鄭南榕的自焚之處自由巷就在我們選區，我為了鄭南榕也要把它選贏！你一想到他做的事情，就會覺得什麼黨內徵召民調都沒什麼大不了的，這次選輸了下次再選嘛！」
